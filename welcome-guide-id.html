Pembaca EPUB Suara

Selamat Datang

Ini adalah aplikasi pembaca eBook dengan fokus untuk membaca nyaring menggunakan perangkat lunak teks-ke-suara (TTS).

Untuk mulai membaca, ketuk tombol 📁 di kiri bawah dan pilih file buku .epub yang telah diunduh di perangkat Anda.

Kontrol

▶ — Mulai/hentikan membaca nyaring
⏪ — Kalimat sebelumnya
⏩ — Kalimat berikutnya
📁 — Buka file
⚙️ — Opsi: suara, kecepatan membaca, dll.
Ketuk di sisi kanan layar — Halaman berikutnya
Ketuk di sisi kiri layar — Halaman sebelumnya
Gerakan mencubit — Sesuaikan ukuran font
